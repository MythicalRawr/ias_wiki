## <img src="../../.gitbook/assets/shared.png" width="32" height="32" /> string.format

```lua
string string.format(str, ...)
```

Formats the given string with the given params<br>Ex: string.format("hello {}, age {}", "world", 20) = "hello world, age 20"<br>
