## ![](images/shared.png "SHARED") json.encode

```lua
string json.encode(table tlb)
```

> Encodes a table to a json string

### Arguments

| Type  | Name | Description             | Optional |
| ----- | ---- | ----------------------- | -------: |
| table | tlb  | The table to be encoded |          |

### Returns

| Type   |     Description |
| ------ | --------------: |
| string | The json string |
