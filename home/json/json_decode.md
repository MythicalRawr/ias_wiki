## ![](images/shared.png "SHARED") json.decode

```lua
table json.decode(string json)
```

> Decodes a string to a lua table

### Arguments

| Type   | Name | Description                   | Optional |
| ------ | ---- | ----------------------------- | -------: |
| string | json | The json string to be decoded |          |

### Returns

| Type  |            Description |
| ----- | ---------------------: |
| table | The decoded json table |
