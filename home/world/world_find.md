# world:find

```lua
Entity world:find(number id)
```

> Finds an entity in the world by id

## Arguments

| Type | Name | Description               | Optional |
| ---- | ---- | ------------------------- | -------: |
| num  | id   | The entity id to be found |          |

## Returns

| Type                                     |      Description |
| ---------------------------------------- | ---------------: |
| [Entity](../../wiki/world/entity\_base/) | The found entity |
