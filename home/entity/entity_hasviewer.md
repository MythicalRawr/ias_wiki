# Entity:hasViewer

```lua
boolean Entity:hasViewer(Player ply)
```

> Returns true if the given player is a viewer

## Arguments

| Type                                      | Name | Description         | Optional |
| ----------------------------------------- | ---- | ------------------- | -------: |
| [Player](../../wiki/entity/player\_base/) | ply  | The player to check |          |

## Returns

| Type    |               Description |
| ------- | ------------------------: |
| boolean | If the player is a viewer |
