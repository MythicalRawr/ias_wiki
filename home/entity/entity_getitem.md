# Entity:getItem

```lua
Item Entity:getItem(number slot)
```

> Returns an item on that slot (nil if none)

## Arguments

| Type   | Name | Description              | Optional |
| ------ | ---- | ------------------------ | -------: |
| number | slot | The slot to get the item |          |

## Returns

| Type                                  |          Description |
| ------------------------------------- | -------------------: |
| [Item](../../wiki/entity/item\_base/) | The item on the slot |
